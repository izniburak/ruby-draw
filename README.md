# ruby-draw
Ruby Draw script for FIFA &amp; PES fans.. Happy tournaments!

## Usage

Clone this repo.

```
$ git clone https://github.com/izniburak/ruby-draw.git
$ cd ruby-draw
```

And run script:

```
$ ruby draw
```
